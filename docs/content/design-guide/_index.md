# Introduction

![OpenStack](/assets/images/OpenStack-Logo-Vertical.svg)

## Welcome to Genestack

[Genestack](/) is a new way of deploying [OpenStack](https://openstack.org)
and [Kubernetes](https://k8s.io) together to create a new type of unified cloud
infrastructure. Before diving into the Genestack details and inner workings, it
is important to decide how you want to structure your cloud.

![Kubernetes](/assets/images/kubernetes-stacked-color.svg)

This section of the documentation covers some basic OpenStack cloud design principles, and offers insight into how these can be realized when building a solution using OpenStack and Genestack.

> [!IMPORTANT]
>
> Watch for **Genestack** boxes like this one to show where various design decisions, technologies, or ideas that Genestack is using!
