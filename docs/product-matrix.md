All release notes are automatically generated using the **Python script** found in [docs/scripts/generate_product_matrix.sh](https://github.com/rackerlabs/genestack/docs/scripts/generate_product_matrix.sh).

To manually generate and update this file, run the following commands from the **root of the repository**:

```shell
pip install -r doc-requirements.txt -r dev-requirements.txt
python scripts/generate_product_matrix.py
```
